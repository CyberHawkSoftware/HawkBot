New Commands Added

Just a friendly reminder that I added 3 new commands! Kanji, Server Info and User Info have been added.
March 29, 2017

Notice

< is the default prefix. When using a command, do not include the <>. They are placeholders for what you would enter.
An alias can be used in place of a command name, and would be used exactly like it would as if it were the command name.

anime

Searches for info on an anime using Kitsu.io as a provider.
Animeclose

<anime <anime>
<anime <anime> --list
