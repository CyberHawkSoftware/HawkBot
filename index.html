The bot responds almost instantly. I have revised and optemized the code in the back end. The commands that take the longest are held up by Discord itself loading their proxied images!

User Tested

As a developer, I test the commands when I make them. The user base does the real testing. The Jisho command was tested on a server of just about 1000 people. It has been refined over the course of over 7 months!

settings

Easy to work with

I designed every command with at least one alias. Believe it or not, that seems like a rare feature. Why type out "help" if you only need a question mark? If you still want to use "help" you still can. For certain commands like for random cat and dog images, you can use "neko" and "inu", or even "猫" and "犬". Most of the commands have a one to two character alias as well.
